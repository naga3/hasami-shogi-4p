# 4人はさみ将棋

> こんなゲーム見たことない。思考ゲーム

1990年代に雑誌へ投稿した MSX2/MSX2+（VRAM 128K）用 BASIC プログラム「4人はさみ将棋」を、作者本人が HTML + JavaScript で復刻したものです。ブラウザだけで動きます（ビルド不要・外部依存なし）。

## 遊びかた

- 12×12 の盤に、青（上）・緑（左）・黄（下）・赤（右）が各 10 枚。青→緑→黄→赤の順に手番が回る
- コマは飛車と同じくタテヨコ一直線に動ける（他のコマは追い越せない）
- 動かしたコマと自分のコマで、他のプレイヤーのコマをタテかヨコにはさむと取れる（列になっていれば複数枚まとめて取れる）
- コマが残り 1 枚以下になると敗退。最後まで生き残った 1 人の勝ち
- 自分の番のときは盤の周辺色が自分のコマの色になる（原作再現）
- **G キー**または「パス」ボタンでパスできる（原作再現）

## モード

原作にあったモードはすべて再現しています。

| モード | 内容 |
|---|---|
| 寝返りモード（NEGAERI） | 取ったコマが消えずに自分のコマになる |
| ダブルスモード（Doubles） | 対面のプレイヤー（青&黄、緑&赤）が味方になり、味方のコマでもはさめる |
| COM 鬼（トレースモード） | そのCOMが通った跡すべてに自分のコマが置かれる恐怖のモード。原作いわく「3人がかりでもほとんど勝てなくなります」 |

さらに本復刻版では画面を 2 種類から選べます。

- **オリジナルモード** — MSX2 SCREEN8 風のディザ盤・丸コマ・PLAY文風のMML効果音
- **アレンジモード** — ネオン系グラフィック + Web Audio 生成のBGM（外部音源ファイルなし）

## COM の強さ

原作の COM は「メチャ弱い」（ほぼランダム）でしたが、本復刻版では思考ルーチンを強化しています。

| レベル | 思考 |
|---|---|
| COM よわい | 原作風。取れる手があれば取る、なければランダム |
| COM ふつう | 捕獲価値 + 次の手番の敵からの反撃リスクを評価 |
| COM つよい | 上記に加え、全敵の反撃リスク（手番が近い敵ほど重視）・敵の敗退・次手の捕獲チャンス・位置・機動力を評価する2手読み |

## 開発

```
# ルール・AI のシミュレーションテスト（AI同士で最後まで対局させる）
node test/sim.js
```

| ファイル | 内容 |
|---|---|
| `engine.js` | 盤面ロジック + AI（純粋ロジック、Node でも動く） |
| `ui.js` | 画面・入力・進行 |
| `audio.js` | MML プレイヤー（原作 PLAY 文風）+ アレンジBGM/SE |
| `style.css` | オリジナル / アレンジ両テーマ |

## 原作との差分

- 盤サイズ（12×12）・初期配置（各辺10枚、四隅空き）・手番順・ルールは原作どおり
- COM に強いレベルを追加（原作相当は「COM よわい」）
- ダブルスモードは原作では「おまけ」（専用思考ルーチンなし・勝者表示1人）だったが、本版はチーム勝利として扱い、味方のコマは取れないようにした
- 膠着防止として、100手連続で捕獲がない場合は残り駒数で決着する（原作にはない）
- 移動アニメーション・待った（undo）を追加
